Java

ICS 111

This course is an overview of the fundamentals of computer science emphasizing problem solving, algorithm development, implementation, and debugging/testing using an object-oriented programming language.

This course used Java as the programming language. The course required the use of jGrasp for the entire semester: JGrasp Website

Github respository for code example: BookStore

This project required utilizing arraylists, methods(constructor, toString, accessor and mutator), and exceptions. The primary challenge was to understand how each class worked with each other.

Please note: BookStore is a stand alone project focusing on arraylists. BookStore2.java was created by the instructor. However, in order to see the Book and BookException files work, BookStore2 is required.

ICS 211

This course reinforces and strengthens problem-solving skills using abstract data types and introduce software development practices. Emphasize the use of searching and sorting algorithms and their complexity, recursion, object-oriented programming, and data structures.

The following files below emphasized creating code that uses existing files provided by the professor. These projects provided experience with working with or modifying existing code as would be the case in real world environments.

Github repository for code example one: Sort Array Example

This project required initializing and storing information from the file input.txt into an array then sorting the information.

Github repository for code example two: CompareTo Example

This project required initializing and storing information from the file compass.csv into an array, comparing values, and modifying data in the array. Each part required the output be posted. This project required the professor's file sorting.java.

Android

ICS 110

ICS 110 placed an emphasis on flowcharting, documentation techniques to create android mobile applications.

This course utilized MIT's App Inventor Built-in Blocks: MIT App Inventor Website

Github repository for code example: Concentration Game!

This featured project was the final for the course. The challenges required to figure out to program cards to interact with each other. If the picture is not the same or same, the next step would to flip the cards back or hide the cards from gameplay.
